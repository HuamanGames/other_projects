Algorithms: Questions and Problems

Why use computers to sort numbers for us?

Human Brains are more efficient than computers but computers
can complete tasks when told to specifically do so. When the
code for the algorithm is executed, then the computer will start
sorting or doing what it was told to do in the code. Many times,
this can be quicker than a human brain sorting all the numbers.

How can sorting data help us in our everyday lives?

Sorting data can help us when we need to organise numbers
automatically and quickly. It can speed up typing, if there are
many numbers. It can also keep information in a neatly stored
way by organising all of the data.

No execution!:

If you gave a computer a list of numbers to sort but didn’t tell it
how to complete the task then it wouldn’t know to start. The data
or code will be stored on the computer but not executed.
