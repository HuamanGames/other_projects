Algorithms

Types of Algorithms:

Four commonly used sorting algorithms are Bubble sorts,
Bucket sorts, Shell sorts and Insertion sorts.

Bubble Sort Algorithm

A bubble sorting algorithm compares two numbers that are side
by side and checks if they are in or out of the order asked for,
for the final result. It will repeat this numerous times and will
finally come to the organised result. When the algorithm has
finished sorting all the numbers it is called a “pass”.

A bubble sort algorithm will be used to make a simple easy
computer program that will tell the computer to check if
numbers are in order and swap them if needed.

A bubble sort algorithm will not be used if there are lots of values for
the computer to bubble sort, because the computer will take a while to
process the task and check all the numbers, although it probably will be
quicker than a human brain.

Bucket Sort Algorithm

Two important pieces of information do you need before you can
start a bucket sort are the amount of values you have and which
they are and the bucket ranges you are going to use to sort the
values.

A bucket sorting algorithm will sort values given to the computer into
categories (into buckets) that can be either a range of numbers (e.g. 4
would go into the range 1-10) or a string category (e.g. there are two
buckets: “colours” and “names”. The colour “blue”, that would have
to be defined earlier and told that it would go into the “colours”
bucket, would go into the bucket called “colours” and not “names”)
and will then take them out of those categories and bubble sort them.

The bucket sorting algorithm find the upper and lower bounds in the
list finding the bucket containing the number range with the highest
and lowest values.

Insertion Sorts:

An insertion sorting algorithm is very similar to the bubble sort but
will start with one value and then will compare the following values
that it adds on (e.g. a list of two numbers: 15, 12. Start with 15.
Then the next number in the list is 12. The computer will insert
this number before the 15 because it 12 less than 15).
